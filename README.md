## 職務経歴(概要)

2018年新卒でBEENOS株式会社にエンジニアとして入社。スマートフォンアプリの開発、および、バックエンドのシステムリプレースを実施。新卒やインターンの採用業務に携わる。

2019年6月にBTM株式会社に転職。同年10月よりプロジェクトリーダーを勤め、翌年4月からグループ長を勤め4人の部下を束ねる。フロントエンドを中心としつつ、PHP・Python・Goなどを用いた幅広い案件を実施。とあるプロジェクトで20名規模のエンジニアをまとめる。中途採用業務の実施、および技術教育を実施。3つの案件を並行し要件定義から開発まで従事。

2021年1月よりヤフー株式会社に転職。実装業務を中心にしつつ、同年9月より個人情報保護法に伴う部署横断プロジェクトのPMを務める。以降の案件も全ての案件でPMを務め、複雑なシステムを円滑なリプレースを実施。2022年3月に50名以上所属している部署内で非公式ながらMVPを獲得。同年6月に個人で運営していたメディア事業を売却。個人で運営したサービスのユーザーが7000人を突破。

2023年2月にフリーランスとして独立。スタートアップ企業にて開発・PM業務に従事。同年12月に委託企業先のVPoEが退職した関係上、同企業の技術責任者を務める。属人化していたサービスをイチからキャッチアップし、新規事業の構築、エラー検知・インシデントフロー体制構築、QAチームの発足、リリースの定量化、デリバリー体制構築、ジュニアエンジニアの育成、CTOクラスのエンジニア採用に携わり委託先の企業はM&Aでグループインを果たす。また並行して2023年4月より中小企業のDX支援を行い、老朽化したシステムの検査、方針の提案を実施。

## 意欲・興味

- 技術者でありながらビジネスとして何が適切な判断になるかを考えます。そのため意思決定の裏側の情報を取りに行く動きを積極的に行います。
- プロダクト開発では少人数チームで小さく早くリリースし、ユーザーからのフィードバックを受けながら改善するアジャイルな開発スタイルを好みます。
- 人為的ミスが出た際には、人ではなく仕組みを疑います。本質的な課題を検討してそれを解決する仕組みを考えることを好みます。

## 強み・今後の改善点

- 強み
  - キャッチアップの速度は早いと言われます。手戻りが少なく開発速度とコード品質については褒めていただくことが多いです。
  - ヤフー株式会社を始めとしてトラフィックが多い規模のサービスを扱うことが多かったため大規模サービスにも一定の知見があります。
  - 状況整理・課題特定・対策プランの構築が得意です。現実的かつ効果的な仕組みづくりには自信があります。
- 今後の改善点
  - 課題を解決する動きは得意ですが、より組織を前進させるようなビジョンを描く能力は改善の余地があると感じます。
  - プロジェクトを回す経験は豊富ですが、プロダクトをグロースさせる経験はまだ課題が残ると感じます。
